# Lager Plan

Простой локальный web-инструмент для тимлида склада:
- сотрудники (добавить/удалить)
- типы задач (из справочника, добавить/удалить)
- план на день (фильтр, задачи, часы)
- план на неделю
- расчет загрузки по общим доступным часам
- печать плана на конкретного сотрудника

## Запуск локально

```bash
cd "Lager Plan"
python3 -m http.server 8080
```

Открыть в браузере: `http://localhost:8080`

## Где хранятся данные

Данные сохраняются локально в браузере (`localStorage`, ключ `lager-plan-v1`).

## Что уже реализовано

1. Базовые настройки смены (сотрудники в смене считаются автоматически из списка, задаются часы в день).
2. CRUD сотрудников.
3. CRUD типов задач.
4. CRUD задач на день.
5. Фильтры задач дня.
6. Печать листа задач для одного сотрудника.
7. CRUD задач на неделю.
8. Расчет доступных/требуемых часов с проверкой дефицита.
9. Разбивка дневной загрузки по типам задач.
10. Персональный контроль дневной загрузки по выбранному сотруднику.
11. Выбор времени задач из фиксированного списка (15 мин ... 9 часов).
12. Статус задачи `Готово` (перечеркивание, перенос вниз списка) и массовое удаление выполненных задач за день.
