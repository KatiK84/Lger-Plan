# Lager Plan

Простой локальный web-инструмент для тимлида склада:
- сотрудники (добавить/удалить, пол)
- план на день (фильтр, задачи, часы)
- план на неделю
- расчет загрузки по правилам `м/ж/оба`
- печать плана на конкретного сотрудника

## Запуск локально

```bash
cd "Lager Plan"
python3 -m http.server 8080
```

Открыть в браузере: `http://localhost:8080`

## Где хранятся данные

Данные сохраняются локально в браузере (`localStorage`, ключ `lager-plan-v1`).

## Что уже реализовано

1. Базовые настройки смены (мужчины, женщины, часы).
2. CRUD сотрудников.
3. CRUD задач на день.
4. Фильтры задач дня.
5. Печать листа задач для одного сотрудника.
6. CRUD задач на неделю.
7. Расчет доступных/требуемых часов с проверкой дефицита.
